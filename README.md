# n8n + LLM — Text Analyzer

This project demonstrates an **end-to-end text analysis automation** using **n8n** (workflow automation tool), **Google Gemini (LLM)**, and a **React.js frontend**.  
The application allows users to input a paragraph and receive either a **summary** or **sentiment analysis** of the text generated by an AI model.

---

## Overview

- **Frontend (React):** Collects user input and sends it to the n8n webhook.
- **Backend (n8n):** Receives the request, calls the Gemini model for text analysis, and returns the response.
- **LLM Integration:** Uses the **Google Gemini node** in n8n (LangChain-based) for natural language processing.

---

## Architecture

**Data Flow:**

### n8n Workflow Steps

| **Node**                   | **Description**                                                                  |
| -------------------------- | -------------------------------------------------------------------------------- |
| **Webhook Trigger**        | Accepts POST requests from the frontend containing `{ text, mode }`.             |
| **Workflow Configuration** | Sets default parameters such as `analysisType` and `model`.                      |
| **Message a Model**        | Sends the input text to **Google Gemini** to generate a summary or sentiment.    |
| **Format Response**        | Formats the output JSON with fields like `analysis`, `success`, and `timestamp`. |
| **Return Results**         | Sends the formatted response back to the frontend.                               |

---

## Frontend (React)

A simple and clean user interface built with **React.js** and **Axios**.

### Features

- Input a paragraph
- Choose between:
  - **Summary**
  - **Sentiment Analysis**
- Displays AI-generated output dynamically
- Handles errors and loading states gracefully

### Example UI Flow

1. User enters text
2. Selects “Summary” or “Sentiment”
3. Clicks **Analyze**
4. Receives formatted LLM output from n8n

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/Krishna0723/plivo
cd text-analyzer
```

npm install

### Create a .env file in the root of your React project:

- REACT_APP_n8nLink=https://your-n8n-instance/webhook/text-analysis

### Then start the app:

- npm start
